# BattleGrounds

一款基于 Godot 4.6 开发的卡牌对战游戏，灵感来源于《炉石传说》酒馆战棋。

## 游戏特色

- **多种族系统**：海盗、元素（火、水、土、风）等多种种族
- **丰富的随从效果**：战吼、亡语、复仇、定时等独特机制
- **商店经营**：购买随从、升级酒馆、管理铸币
- **策略对战**：合理的随从站位与效果搭配

## 技术栈

- **引擎**：Godot 4.6
- **渲染**：Forward Plus
- **物理**：Jolt Physics
- **分辨率**：2560 x 1440

## 游戏机制

### 随从属性

- **攻击力**：随从攻击时造成的伤害
- **生命值**：随从可承受的伤害
- **等级**：1-6级，影响商店刷新概率

### 种族特性

#### 海盗
- 专注于铸币获取与经济优势
- 通过购买和出售随从获得额外收益

#### 元素
- **火**：攻击性强，擅长造成伤害
- **水**：辅助型，获取与生成元素牌
- **土**：防御型，生命值较高
- **风**：拥有圣盾、风怒等特殊能力

### 特殊效果

| 效果 | 描述 |
|------|------|
| 战吼 | 打出时触发效果 |
| 亡语 | 死亡时触发效果 |
| 复仇 | 指定数量的友方随从死亡后触发 |
| 定时 | 每隔指定时间可点击触发 |
| 圣盾 | 免疫第一次伤害 |
| 风怒 | 每回合可攻击两次 |
| 嘲讽 | 敌方必须优先攻击此随从 |
| 磁力 | 可吸附在机械和元素上 |

## 快速开始

### 环境要求

- Godot 4.6 或更高版本

### 运行方式

1. 克隆仓库
```bash
git clone https://github.com/comipainter/battle-grounds.git
```

2. 使用 Godot 打开项目
```bash
godot --path battle-grounds
```

3. 按 F5 或点击运行按钮开始游戏
